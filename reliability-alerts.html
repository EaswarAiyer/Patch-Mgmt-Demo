Automated decisions

Auto-decline low-reliability patches

Applies organization-wide. Every patch scoring below the threshold is declined automatically across all groups and excluded from deployment. Individual patches can still be overridden manually.

How automation interacts with reviewAuto-decline is a single organization-wide rule — turning it on affects every group. Automated decisions are logged in the alert list and tagged as “Automation”, and any admin can override an auto-decision manually.
